# CIC Knowledge Base

Welcome to the CIC Knowledge Base repository. This repository stores all documentation, scripts, configurations, and integration test runners for the NotebookLM Synchronization Pipeline.

## Repository Layout
- `docs/kb/notebooklm-sync/`: Synchronization documentation (Architecture, Operator Rules, boundaries, etc.).
- `scripts/notebooklm/`: Execution automation and background Git hooks.
- `tests/`: Automated integration verification tests.
- `pyragify.yaml`: Codebase packing filter settings.
